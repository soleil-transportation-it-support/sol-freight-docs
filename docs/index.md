# Soleil Freight BRD Portal

This portal is generated by a local-first automation pipeline for Ocean Export PoC.

## Scope

- Module: Ocean Export
- Flows: MBL and HBL
- Output channels:
- Internal HTML site (MkDocs / GitHub Pages)
- Local PDF package for customer sharing

## Pipeline Stages

1. Reset and seed backend data.
2. Crawl UI screens with Playwright.
3. Generate BRD markdown from scan artifacts.
4. Build static docs and optionally export PDF.
